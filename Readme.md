<h1> Employee Management </h1>

This Project is about Employee management 

Feature
-----
-> Add Employees.
-> Listing Employees.
-> Inline employee info Edit.
-> Export CSV.


Requirement
-----------
HTML 5
JS/JQuery
PHP 7.2 or higher
MySQL 5.6



Setup
-------
-> Import 'employees_db.sql' in phpmyadmin.
-> Change Your configuration in config.php
-> Run 'http://localhost/projectfolder/'
